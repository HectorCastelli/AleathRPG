Dados

Aleath utiliza D6 para determinar sucessos e falhas. Ao rolar um D6, considera-se sucesso os números 4,5 e 6.

Para executar uma ação o jogador precisa descrevê-la completamente e o mestre de jogo analisa a dificuldade da mesma, determinando o número de sucessos necessários para que a ação seja bem suscedida.

O mínimo de dados que um jogador rola é 2. Para cada característica, habilidade, perícia e estado favorável a ação descrita você recebe mais um dado para rolar. Para elementos negativos, se perde um dado.
